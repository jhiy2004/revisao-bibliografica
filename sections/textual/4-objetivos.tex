\section{Objetivos}

\subsection{Objetivos principais}
Desenvolver um sistema para simulação das equações de águas rasas, usando Devito para
solucionar as equações de forma numérica e Vulkan para visualizar o resultado da simulação.

\subsection{Objetivos especificos}

\begin{itemize}
    \item Implementar um solucionador numérico das equações de águas rasas em 2D
        utilizando o Devito.
    \item Avaliar o desempenho do sistema em termos de quadros por segundo
    \item Validar o sistema com cenários de referência (ondas radais e
        propagação de ondas costeiras).
\end{itemize}
